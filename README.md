# Teachers_Hub
A final project on my ADV101,  with a tech stack of: Next.js – Frontend Tailwind CSS - CSS Framework Node.js - Backend MySQL - Database

This project is for teachers to tally their work time every time they clock in,
in a room to teach on which an admin post's a task that is assigned or not assigned to them that they claimed,
in which an admin has full control

The project has CRUD Functionalities
Create – Insert data into the database
Read – Display or retrieve existing data.
Update – Edit or modify existing data.
Delete – Remove records from the database
